Introduction to HTML by Mairead Crowe

Doctype

Description:

A doctype or document type declaration is an instruction that tells the web browser about the markup language in which the current page is written

Example of the code:

<!doctype html>

HTML

Description:

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for Web pages. HTML elements are the building blocks of HTML pages.

Example of the code:

<html></html>

Head

Description:

The head is the first section in the code containing information about a web page's properties and links to external related files.

Example of the code:

<head></head>

Body

Description:

The HTML body tag defines the main content of the HTML document or the section of the HTML document that will be directly visible on your web page.

Example of the code:

<body></body>

Title

Description:

The title tag defines a title of an HTML document. A title tag must be the precise description of a page’s content. The title is displayed in the browser toolbar and on search engine results pages as a clickable headline.

Example of the code:

<title></title>

P

Description:

The p tag defines a paragraph.

Example of the code:

<p></p>

Working Example:

This is an example of a paragraph

Span

Description:

Span is an element which is used to color a part of a text. It is an inline container used to mark up a part of a text, or a part of a document.

Example of the code:

<span></span>

Working Example:

My mother has blue eyes.

 

Div

Description:

The HTML division tag, called "div" for short, is a special element that lets you group similar sets of content together on a web page. You can use it as a generic container for associating similar content.

Example of the code:

<div></div>

Working Example:

This is a heading

This is a paragraph.

 

Heading Elements - h1-h6

Description:

HTML headings are defined with the h1 to h6 tags. H1 defines the most important heading. H6 defines the least important heading. Note: Browsers automatically add some white space (a margin) before and after a heading. Search engines use the headings to index the structure and content of your web pages.

Example of the code:

 <h1>Heading 1</h1> 

<h2>Heading 2</h2>

 <h3>Heading 3</h3> 

<h4>Heading 4</h4> 

<h5>Heading 5</h5>

 <h6>Heading 6</h6>

Working Example:

 

 

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Formatting Elements - strong, emphasis

Description:

HTML contains several elements for defining text with a special meaning. Such as: b - Bold text strong - Important text i - Italic text em - Emphasized text mark - Marked text small - Smaller text del - Deleted text ins - Inserted text sub - Subscript text sup - Superscript text

Example of the code:

<b></b>

Working Example:

This text is bold

Tables & Caption

Description:

The table tag defines an HTML table. An HTML table consists of one table element and one or more tr, th, and td elements.The caption tag defines a table caption. The caption tag must be inserted immediately after the table tag.

Example of the code:

<table></table><caption></caption>

Working Example:

Name Surname Course
Mairead Crowe Web Authoring

Images & Alt

Description:

The required alt attribute specifies an alternate text for an image, if the image cannot be displayed.The alt attribute provides alternative information for an image if a user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).

Example of the code:

<img src="images/Alt.jfif" width="474" height="320" alt=""/>

Links

Description:

An HTML link is used to link other web pages to a document. When users click on links on web pages, they are brought to other web pages or websites. An HTML document contains many links, which are also known as hyperlinks. Web developers can link from the text, images, and other types of media.Links are specified in HTML using a tags.

Example of the code:

<a href="url">link text</a> 

Working Example:

Visit RTE!

Quote & Blockquote

Description:

Quote tag in HTML is used to show short quotations using q tag. It is enclosed in between q …. /q. It helps to indicate that the contained text is in the inline quotation.Blockquote tag is used to define a block of text which is quoted from another source. The Browser usually displays the content within blockquote tag as indented text.

Example of the code:

<q></q>

<blockquote></blockquote>

Working Example:

This is a quote
This is a blockquote

Description:

The abbr tag in HTML is used to define the abbreviation or short form of an element.

Example of the code:

<abbr></abbr>

Working Example:

Special Symbols and Character entities: Copyright, Trade Mark, Registered Trademark, Ampersand, Euro, Non-breaking space, Less than, Greater than

Description:

Characters with special meaning in HTML are called reserved characters. For example, left and right angle brackets are reserved in HTML to identify the opening and closing tags of elements. Characters that don’t appear on your keyboard include things like the copyright symbol (©) and the mathematical value pi (π).

Example of the code:

<p>I will display &euro;</p>

Working Example:
